This repository uses the same dataset and classification problem as the author’s other project, [ChestXRay-NeuralNet-Classifier](https://github.com/aminbakhshi00/ChestXRay-NeuralNet-Classifier). That project uses pure dense layers with no CNN, while this repository uses a CNN-based approach. The CNN model achieved an F1 score of **0.598**, compared to **0.34** for the dense-only model.
