This repository addresses the same dataset and classification task as the author’s other project, [ChestXRay-NeuralNet-Classifier](https://github.com/aminbakhshi00/ChestXRay-NeuralNet-Classifier). That version uses only dense layers and no CNN architecture, while this repository uses a CNN-based model. The CNN approach achieved an F1 score of **0.598**, compared to **0.34** for the dense-only model.
